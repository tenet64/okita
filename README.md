# **okita!!**

## **サービス概要**

**okita!!** は、早起きを「自分の意思」ではなく 「つながり」と「仕組み」で実現するデュアルモード型の起床支援サービスです。

ソロモード

個人の習慣化支援。 起床時間設定 ＋ ワンタップ認証（MVP）で、無理なく継続できる習慣化をサポートします。

マルチモード（みんなで早起きチャレンジ）:

オープンな「約束」による共同起床。 掲示板で早起きチャレンジを公開-参加し、募集人数が揃うと即座にマルチが成立。メンバー全員での同時刻の起床証明を必須とすることで、寝坊しづらいプレッシャーや達成感を提供します。


## **このサービスへの思い・作りたい理由**

私は、目覚ましアプリをどれだけ工夫しても「スヌーズ → 二度寝」を繰り返してしまうタイプでした。

しかし、友人と「朝ジムに行こう」と約束した日は絶対に起きられる経験から、

 人は“自分のためには弱いが、他人との約束には強い”
 

ということを実感しました。

そこで

- 一人でも続けられる仕組み
- 誰かと約束することで続けやすい仕組み

この２つを組み合わせた、軽量なのに強制力のあるサービスを作りたいと思い、okita!! を着想しました。

## **ユーザー層について**

| **ユーザー層** | **特徴** | **対象にした理由** |
| --- | --- | --- |
| **ソロモード利用者** | 早起きしたいが、誰かと一緒はストレスに感じる人 | 「ワンタップ起床認証」「ゆるいペナルティ」で継続しやすいため |
| **マルチモード利用者** | 誰かと一緒に頑張りたい人（友達や恋人） | “人の目”の効果が強く、寝坊しにくくなる |


## **サービスの利用イメージ**

### 共通
新規登録 → モード選択: 「ソロで始める」または「チャレンジに参加/作成」を選択。

### ソロモード
起床目標を設定: 起床時間を入力。

翌朝：起床ページで起床証明: ワンタップ（MVP）またはで起床を証明。

結果: 成功 → ポイント＋ / 失敗 → ポイント−

### マルチモード（みんなで早起きチャレンジ）
**チャレンジ作成/参加:**

- 作成: 起床目標時間-募集人数などを設定し、掲示板に投稿。

- 参加: 掲示板の投稿から「一緒におきたいボタン」をクリックし、仮参加。

**マルチ成立（即時）:**

参加人数が募集人数に達した時点で、自動的にチャレンジグループが即時成立。

翌朝：共同起床証明（MVP＝長押し認証）:

目標時間になると、成立メンバー専用の**「起床ページ」**へ遷移。

全員が同時刻（±5分以内など）に画面をタップし、起床を証明。

成功するとリアルタイムで相手に反映：「パートナーが完了しました ☀️」

### バトルモード（1対1）

**体内時計の正確さ**を競うバトル。
 バトルページにて募集に参加または作成する
 
 1対1が確立したらバトルルームへ移行

- **ルール（例:7時起床の場合）**
- 7時にお題：「**7:00:10** ジャストで止めろ！」と表示される（ここの秒数はランダムの仕様）
- 画面の時計表示が消える（ブラインド状態）。
- ユーザーは頭の中で10秒数え、ボタンを押す。
- ターゲットタイムとの誤差（ミリ秒）が少ない方が勝ち。

### みんなのステータス画面:

相手の起床状況（タップ完了/未完了）

みんなの達成率、過去7日ログ
### ポイント制度
- GitHubの草のようなイメージ
- ソロモード　1回起床成功毎に1ポイント
- マルチモード　集まった人数×ポイント　※“全員がタップ完了しなければ、誰もポイントを得られない” 。
- バトルモード　買った方が１ポイントゲット
### バッジの付与
ポイントを貯めることや、連続起床回数など条件に応じてバッジを獲得できる。（バッジはユーザページで確認可能　※これもGitHubのバッジを参考に発案）

例）50ポイントバッジ、100ポイントバッジ、連続記録5日目、ソロ達成10回目、バトルの猛者など

## **ユーザーの獲得について**
- SNS連携：「今から早起きチャレンジ募集！」の内容をSNSへワンタップでシェアできるように設計し、新たな参加者を募る導線を作る。
- SEO戦略：	「二度寝防止」「習慣化アプリ ストレスフリー」「目覚まし 継続」など、課題解決に直結するキーワードを重視。

## **サービスの差別化ポイント**
###  競合サービス比較表

| サービス | つながり |  習慣化へのプレッシャー |  リアルタイム性 |  ターゲット課題 |
| :--- | :--- | :--- | :--- | :--- |
| **標準アラーム** | ソロ | **低い** (音のみ) | **なし** | シンプルな起床通知 |
| **みんチャレ** | チーム (5人程度) | **中** (事後報告の義務) | **低い** (当日中報告可) | 継続的な習慣化 |
| **レッツ朝活サロン** | 大規模コミュニティ | **中** (会費・イベント参加) | **中** (イベントはzoomで同期) | 知識獲得、モチベーション維持 |
| **モーニングコール** | 1対1 | **高い** (直接的な人為的介入) | **高い** (正確な時間で同期) | 確実な起床 (保険的役割) |
| **okita!!** | **1人~** | **高い** (連帯責任) | **高い** (同時刻アクション必須) | **スヌーズ癖の軽減、二度寝防止** |

okita!!は、この中でも特に「連帯責任の高さ」と「同時刻認証」、「ゲーム性」という点でユニークな機能を持っています。

## **機能候補**
### **MVP** 
- ユーザー認証（Devise）
- 起床ログ記録、起床履歴画面（達成率-前日ログ）
- **ソロモード** 起床設定＋ワンタップ起床証明 
- **マルチモード** 掲示板へのチャレンジ投稿、仮参加（ボタン）、募集人数到達による**自動成立**
- **長押しによる共同起床証明**、起床状況リアルタイム反映（Hotwire）

- ポイント特典でバッジ付与
